# askuniversal.netlify.app

Large, multi-page corporate website for ASK LAW FIRM (Tax, Valuation & Financial Advisory). Built as a static site with direct-to-HTML content editing and organized content sections for services, industries, resources, and careers.

**Live Website**
```
https://www.askuniversal.com/
```

**Project Highlights**
- Multi-section corporate site with dedicated areas for services, industry focus, thought leadership, and recruiting
- Static-first delivery for reliability, speed, and easy Netlify deployment
- Sitemap generation integrated in Netlify configuration

**Project Layout**
- Root HTML pages: `index.html`, `services.html`, `who-we-are.html`, `who-we-serve.html`, `careers.html`, `contact-us.html`, and other top-level pages
- Content sections: `resources/`, `services/`, `who-we-serve/`, `careers/`, `360-solutions/`
- Shared assets and vendor folders: `assets/`, `themes/`, `modules/`, `plugins/`, `combine/`


**Deployment**
Netlify publishes directly from the repository root. The build configuration is in `netlify.toml`.

**Sitemap**
The sitemap is generated by `@netlify/plugin-sitemap`. The configuration in `netlify.toml` sets:
- `baseUrl` to `https://askuniversal.netlify.app`
- `buildDir` to `.`
- `prettyURLs` to `true`
- Exclusions for assets, vendor folders, and duplicate `*-2.html` / `*-3.html` pages

**Editing Content**
- Update page content by editing the relevant `.html` files
- Page-specific content is organized under the section folders listed above
- Shared styles and scripts live under `assets/`, `themes/`, `modules/`, and `plugins/`

**Source Code Policy**
This repository is not intended for public code distribution. The live website is published from this private source to protect proprietary content and structure.
